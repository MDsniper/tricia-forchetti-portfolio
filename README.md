# Tricia Forchetti Portfolio

Static portfolio site for a talent acquisition consultant, deployed on Cloudflare Workers.

## Structure

- `dist/` — Static site assets (HTML + images)
- `wrangler.toml` — Cloudflare Workers configuration

## Deploy

```bash
npx wrangler deploy
```

Then visit `https://tricia-forchetti-portfolio.<your-subdomain>.workers.dev`.
